This site was generated using [HUGO](https://gohugo.io) via [Netlify](https://www.netlify.com/). The theme that this site uses was forked from [Eddie Webb](https://github.com/eddiewebb/hugo-resume) and heavily modified to be more efficient as well as to fit my personal needs. Netlify acts as my CI. If I push out a change to the [my-portfolio repo](https://github.com/tofan0810/my-portfolio) on GitHub then Netlify will automatically trigger and rebuild this website for production.
